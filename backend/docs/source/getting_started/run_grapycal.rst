Run Grapycal
==================

1. Run the Grapycal Server

.. code-block:: bash

    python -m grapycal

2. Run the Web Server

.. code-block:: bash

    cd frontend
    npm run app

The web UI is now avaliable at http://localhost:9001/.

.. note::

    In future versions, the web server will be integrated into the Grapycal server.
